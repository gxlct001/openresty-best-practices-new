# 第三部分 OpenResty 和生态周边

这一部分主要介绍 OpenResty 的安装、基本使用、Nginx 的 C 模块以及使用 cosocket API 重新实现的 Lua-resty-* 库。

以下是收集的一些相关教程和书籍列表，欢迎大家贡献，把知识和实践经验分享给更多的小伙伴，共同学习，共同成长。
